# 3-Minute Hackathon Demo Flow

1. Open the landing page and state the problem: random teammates break ranked games before the match starts.
2. Login with the demo account: `demo@clutchq.com` / `demo123`.
3. Open `/games` and show the game hub, filters, and active room cards.
4. Open a game detail page and explain that ClutchQ is game-first, not a generic LFG board.
5. Click **Find Squad Now** and show the animated match breakdown.
6. Expand **Why this match?** on a player card to show role, rank, region, mic, and availability signals.
7. Create or open a lobby and show member fit, ready checks, and the squad chemistry graph.
8. If Discord env is configured, create a Discord voice room and show the invite controls.
9. Open Profile and show Steam identity, library, friends/achievement limitations, avatar upload, and player score.
10. If time remains, open Leaderboards and Admin Dashboard to show activity ranking and moderation.
