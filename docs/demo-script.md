# 3-Minute Hackathon Demo Flow

1. Open the landing page.
2. Explain the problem: random teammates ruin ranked games.
3. Login as demo user: `demo@clutchq.com` / `demo123`.
4. Open the dashboard.
5. Click Find Squad Now.
6. Show Best Squad Found and squad compatibility.
7. Open the Live DNA Matchmaking Visualizer.
8. Expand Why This Match on a player card.
9. Show availability heatmap overlap.
10. Open lobby details.
11. Show squad chemistry graph.
12. Show missing role detector.
13. Send a join request.
14. Login as admin: `admin@clutchq.com` / `admin123`.
15. Open admin dashboard and show analytics plus moderation.
16. End with future scope: real-time chat, Discord OAuth, tournaments, and ML role suggestions.
